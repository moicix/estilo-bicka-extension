# Estilo Bicka Airtable Block

This is an Airtable Block for Estilo Bicka.

## Development

To run this block for testing in Airtable, use the following command:

```bash
block run
```

This will start a local development server. You can then load the block in Airtable by selecting "Add a block" and choosing "From URL" and entering the local server address (usually `http://localhost:9000`).

## Publishing

To publish the extension, use the following command:

```bash
block release
```

This will build the block for production and prepare it for publishing to the Airtable Marketplace.
